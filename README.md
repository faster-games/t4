# Mono.TextTemplating

Fork of the great [mono/t4](https://github.com/mono/t4) project, slimmed down and packed up to be used in Unity3D, specifically via UPM.

## Getting started

In a new Unity project, add the following to `Packages\mainfest.json` under `dependencies`:

```
"com.faster-games.t4": "git+https://github.com/faster-games/t4.git#latest",
```

> Note: `latest` may be replaced with [a specific version](https://github.com/faster-games/t4/releases).
